### Why Defer to Post-MVP?

2. **Market Validation:** Need to prove candidate-side value first
3. **Resource Allocation:** Team focused on core differentiators (AI interview + matching)
4. **B2C Before B2B:** Launch free tier for candidates before enterprise sales

## Planned Features

### Recruiter Analytics (EP4-S2)

- Candidate funnel visualization
- Interview completion rates by role
- Match score distributions
- Time-to-hire metrics
- Cost per hire tracking

### Collaborative Tools (EP4-S4)

- Team member roles & permissions
- Candidate scorecards with comments
- Interview panel scheduling
- Shared candidate notes
- Hiring decision workflows

### Enterprise Integration (EP4-S5)

- Public API with OAuth
- Webhook notifications
- ATS integration beyond Workable (Greenhouse, Lever)
- HRIS integration (BambooHR, Workday)
- SSO (SAML, OIDC)

### Monetization (EP4-S8)

- Tiered pricing (Free, Pro, Enterprise)
- Usage-based billing for AI features
- White-label options
- Custom branding
- SLA guarantees
